元素的分类 具体元素 默认特征
块级元素（block-level elements） 替换元素（replaced elements）
非替换元素（no-replaced elements） div p pre h1-h6 ul ol li dl dt dd table form
article aside footer header hgroup main
nav section blockquote hr等		 独占父元素一行
可以随意设置宽高
高度默认由内容决定
行内级元素（inline-level elements） 替换元素（replaced elements） img input iframe video embed canvas audio object等 跟其他行内级元素同在一行显示
可以随意设置宽高
非替换元素（no-replaced elements） a span strong code label等 跟其他行内级元素同在一行显示
不可以随意设置宽高
宽高→内容决定
